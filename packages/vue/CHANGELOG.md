# @squircle-js/vue

## 0.1.0

### Minor Changes

- [`92fac94`](https://github.com/bring-shrubbery/squircle-js/commit/92fac9461084a363af3a27f321a928919e571d69) Thanks [@bring-shrubbery](https://github.com/bring-shrubbery)! - Introduce `@squircle-js/vue` for Vue 3 (^3.3). Ships three APIs:

  - Components: `<Squircle>`, `<StaticSquircle>`, `<SquircleNoScript>`
  - Directives: `v-squircle`, `v-static-squircle` (the `asChild` replacement)
  - Composables: `useSquircle`, `useStaticSquircle`

  Also ships `SquirclePlugin` for global registration via `app.use(SquirclePlugin)`.
